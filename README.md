# OTCamera-tools

This repository contains multiple helper tools for OTCamera.

The repository is split into multiple packages using `uv`'s [workspace feature](https://docs.astral.sh/uv/concepts/projects/workspaces/).

## Contents

- `packages/otc-metrics`: Tools to help logging metrics to csv files.
- `packages/otc-video-eval`: Small script for evaluating frame counts in h264 files.
